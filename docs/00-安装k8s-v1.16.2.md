# k8s v1.16.2安装说明文档

- 进入ansible_playbooks后，执行ansible-playbook k8s-masters.yml和ansible-playbook k8s-workers.yml后即完成了初始化工作。
当然这个不通用，仅限于我的环境。后续可以将一些基础环境的搭建文档都写出来。

- 在master.k8s.example.com上执行命令：`kubeadm init --kubernetes-version v1.16.2 --pod-network-cidr 10.244.0.0/16`，之后初始化成功，
日志文件为k8s-v1.16.2-install.log，按指示命令进行设置。  
    ```shell
    mkdir -p $HOME/.kube 
    sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config  
    sudo chown $(id -u):$(id -g) $HOME/.kube/config
    ```

- 安装网络插件
    ```shell
    kubectl apply -f k8s_platform/kube-flannel-v0.11.0-amd64.yml
    ```
- 安装七层代理
    ```shell
    kubectl apply -f k8s_platform/ingress-nginx-0.28.0-mandatory-deployment.yaml
    kubectl apply -f k8s_platform/ingress-nginx-service.yaml
    ```
